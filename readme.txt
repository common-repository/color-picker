Add Color Picker on any page Help you to create dynamic color for user front website. 

== Description ==
When we select a color in the backend from the color picker the text in the frontend shows the selected color.

== Installation ==
1. Download plugin and unzip it to your `/wp-content/plugins/` plugin directory.
2. After that go to your plug-in manager and activate it.
3. After activate Color Picker on any page Go to SETTING menu and select color.
